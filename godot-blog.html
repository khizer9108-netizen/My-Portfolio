Why I love Godot for Game Development

June 2026

Godot, a free open-source game engine, is used to create 2D and 3D games. It has its own programming language too, known as GDScript. GDScript is very similar to Python; in fact, I would say that anyone who knows Python can learn GDScript in a day!

I have always wanted to develop my own games. One day, at my school, a friend announced that a game developer hackathon was coming up. This was the moment I could finally try to make games. I must admit, I had very little knowledge about coding and designing, but in today’s world, we have powerful search engines, so it wasn’t an issue. Winning was not my main priority; it was the experience.

As soon as I registered, I had to grab two friends, even though they weren't the most professional at this stuff. When I first opened Godot, it seemed complicated, but after the first week, I got the hang of it. The best part of Godot is that the interface is quite neat compared to other engines. It’s lightweight, so I had no issues on my not-so-powerful laptop.

The best part is that it is open-source—no need to pay any money— so there are no annoying pop-ups asking the user to pay! The node and scene architecture is quite intuitive; for example, you can reuse a scene with just a few clicks.

To sum it all up, Godot is a small yet extremely well-designed game engine that can work on most computers. If you're into game development and have never tried making a game, give Godot a try! You will love it.
